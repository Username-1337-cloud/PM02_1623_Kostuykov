# PM02_1623ИСИП_Костюков

Пример выполнения экзаменационного билета по ПМ.02.

## Состав репозитория

- `docs/use-case-diagram.pdf` - диаграмма вариантов использования приложения.
- `src/TransportNorthWest` - исходный код приложения на C# WinForms.
- `tests/test-scenarios.pdf` - три тестовых сценария и результаты проверки.
- `examples/input.csv` - пример файла с исходными данными.

## Назначение приложения

Приложение решает закрытую транспортную задачу методом северо-западного угла.

Реализованы функции:

- ввод запасов, потребностей и матрицы тарифов вручную;
- загрузка исходных данных из CSV-файла;
- проверка корректности данных;
- расчет плана перевозок методом северо-западного угла;
- вывод хода решения и итоговой стоимости;
- сохранение результата в текстовый файл;
- очистка полей.

## Формат ручного ввода

Запасы и потребности вводятся в одну строку через пробел, запятую или точку с запятой.

Пример запасов:

```text
40 22 38
```

Пример потребностей:

```text
15 25 40
```

Матрица тарифов вводится построчно:

```text
8 6 10
9 7 4
3 4 2
```

## Формат CSV-файла

Первая строка содержит потребности. Каждая следующая строка содержит запас поставщика и тарифы до потребителей.

```csv
15;25;40
20;8;6;10
30;9;7;4
30;3;4;2
```

## Пример результата

Для данных из `examples/input.csv` приложение строит план:

```text
A1 -> B1 = 15
A1 -> B2 = 5
A2 -> B2 = 20
A2 -> B3 = 10
A3 -> B3 = 30
```

Итоговая стоимость перевозок: `415`.

## Запуск

1. Открыть `PM02_1623ИСИП_Костюков.sln` в Microsoft Visual Studio.
2. Запустить проект `TransportNorthWest`.
3. Ввести данные вручную или загрузить файл `examples/input.csv`.
4. Нажать кнопку `Решить`.
5. При необходимости сохранить ответ кнопкой `Сохранить ответ`.
